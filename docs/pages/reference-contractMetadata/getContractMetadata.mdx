# Get Contract Metadata

Get a contract's metadata.

```typescript
client.projectContractMetadata.get.query({
    chainId: 130130,
    address: "0xD7bA09F9787Aa6FF2c41B16c557Ff20d0D2901cB",
});
```

**Required fields**:

-   `chainId`: Id of the chain you selected when creating your project.
-   `address`: Address of the contract (collection).

**Optional fields**: See [Get contract metadata](https://docs-api.owlprotocol.xyz/reference/projectcontractmetadata-get)
