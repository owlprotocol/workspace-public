# Create Project User

Create a new user for a project.

```typescript
const user = client.projectUser.createOrSet.mutate({
    email: "user2@gmail.com",
    fullName: "User 2",
    externalId: "123456789",
});
```

**Required fields**:

-   `email`: User's email address

**Optional fields**: See [Create or set project user](https://docs-api.owlprotocol.xyz/reference/projectuser-createorset)
