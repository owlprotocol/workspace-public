# Get Token Template

Get information (metadata) of a Token Template.

```typescript
client.projectTokenTemplate.get.query({
    id: "6c3d3ffc-1580-47bd-88c6-4c64642c9319",
});
```

**Required fields**:

-   `id`: Id of the Token Template

**Optional fields**: See [Get project token template](ref:projecttokentemplate-get)
