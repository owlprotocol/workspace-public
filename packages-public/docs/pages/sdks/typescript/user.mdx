# Create Project User

Create a new user for a project.

```typescript
const user = client.projectUser.createOrSet.mutate({
  email: "user2@gmail.com",
  fullName: "User 2",
  externalId: "123456789"
})
```

**Required fields**:

- `email`: User's email address

**Optional fields**: See [Create or set project user](ref:projectuser-createorset)

# Get Project User

Get an existing user's information, including Safe Wallet address.

```typescript
const user = client.projectUser.get.query({
	chainId: 130130,
  email: "user2@gmail.com"
})
```

**Required fields**:

- `chainId`: Id of the chain you selected when creating your project.
- `email`,`userId`, or `externalId`: You can query by either of these three options.

**Optional fields**: See [Get project user](ref:projectuser-get)

# List Project Users

List all users from a project.

```typescript
const users = client.projectUser.list.query()
```

**Required field**: None

**Optional fields**: See [List project users](ref:projectuser-list)
