# Create Token Template

Create a Token Template.

```typescript
client.projectTokenTemplate.createOrSet.mutate({
	address: "0xD7bA09F9787Aa6FF2c41B16c557Ff20d0D2901cB", // Optional, but recommended
  metadata: {
  	name: "Template Name",
    description: "Template Description",
    image: "https://picsum.photos/200",
    // Other metadata fields
  }
})
```

**Required fields**:

- At least one metadata field (`name`, `description`, `image`, `attributes`, etc...)

**Optional fields**:

- `address`: We highly recommend passing the address of a collection. This would associate the template to a specific collection.

 See [Create or set project token template](ref:projecttokentemplate-createorset)

# Get Token Template

Get information (metadata) of a Token Template.

```typescript
client.projectTokenTemplate.get.query({
	id: "6c3d3ffc-1580-47bd-88c6-4c64642c9319"
})
```

**Required fields**:

- `id`: Id of the Token Template

**Optional fields**: See [Get project token template](ref:projecttokentemplate-get)

# List Token Templates

```typescript
client.projectTokenTemplate.list.query()
```

**Required fields**: None.

**Optional fields**: See [List project token templates](ref:projecttokentemplate-list)

# Patch Token Template

Modifying certain fields for a Token Template, while keeping the other previously defined fields intact.

```typescript
client.projectTokenTemplate.patch.mutate({
  id: "6c3d3ffc-1580-47bd-88c6-4c64642c9319",
  metadata: {
  	description: "New Token Description" // Replace description with this one
  }
})
```

** Required fields**:

- `id`: Id of the Token Template.
- At least one of the metadata fields (`name`,`description`, etc...).

**Optional fields**: See [Patch project token template](ref:projecttokentemplate-patch)
